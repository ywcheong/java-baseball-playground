# 숫자야구 기능

- 컴퓨터는 게임이 시작하면 무작위로 "정답 플레이트"를 생성한다.
	- "플레이트"는 1~9까지의 숫자 중 서로 다른 3개를 뽑아 만들어진 3자리 숫자이다.
- 사용자는 컴퓨터의 "정답 플레이트"를 알아내기 위해 "질문 플레이트"를 컴퓨터에게 제공한다.
	- 컴퓨터는 "질문 플레이트"와 "정답 플레이트"를 대조해 적절한 결과를 반환한다.
		- 어떤 숫자가 "질문 플레이트"와 "정답 플레이트"의 같은 자릿수에 위치했다면, +1 스트라이크이다.
		- 어떤 숫자가 "질문 플레이트"와 "정답 플레이트"의 다른 자릿수에 위치했다면, +1 볼이다.
		- 컴퓨터는 스트라이크와 볼 각각의 갯수만을 사용자에게 알려준다.
		    - 예를 들어 "정답 플레이트"가 "123", "질문 플레이트"가 "139"라면 1 스트라이크 1 볼이다.
- 사용자의 "질문 플레이트"가 3 스트라이크 결과를 냈다면, 사용자의 승리를 선언한다.
    - 이때 사용자의 점수는 승리에 이르기까지 사용한 "질문 플레이트"의 갯수이다.
- 사용자의 "질문 플레이트"가 유효하지 않다면 사용자에게 오류 메시지를 출력하되, 게임을 종료하지는 않는다.
    - 오류 메시지는 점수를 계산하는 플레이트의 갯수로 합산하지 않는다.
- 사용자의 "질문 플레이트"가 "000"이라면, 사용자의 패배를 선언하고 "정답 플레이트"를 출력한다.
